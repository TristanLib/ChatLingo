## 🎮 **如何运行**

1. **打开项目**
```bash
cd /Volumes/disk2/claude_projects/ChatLingo/ChatLingo-iOS/ChatLingo
open ChatLingo.xcodeproj
```

2. **选择模拟器**
- 推荐：iPhone 15 (iOS 17+)
- 最低要求：iOS 15.0+

3. **运行应用**
- 按 `Cmd + R` 或点击播放按钮
- 应用将显示完整的功能界面
